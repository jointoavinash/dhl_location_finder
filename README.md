# Installation:
1. Place all the content from *DHL_LOCATION_FINDER* folder to `web/` at root level.
2. Enable the module using Drush or Drupal UI: 
`
drush en dhl_location_finder -y
`
3. Visit /dhl-location-finder to use the form.
